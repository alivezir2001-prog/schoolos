# SchoolOS Glossary

## Why This Document Exists

SchoolOS is more than a software project.

It is a shared language.

Before we write code, design databases or build user interfaces, we must first agree on the meaning of the concepts we use.

This glossary defines the common language of SchoolOS.

Every concept described here should have one clear meaning across the entire platform.

If a concept is not clearly defined, it should not yet exist in the software.

---

# Story

## Definition

A Story is the living narrative of the development journey of a person, a school or a community.

A Story is not a report.

It is not a collection of records.

It is the meaningful connection between experiences, observations, achievements, challenges and growth over time.

SchoolOS recognizes that every student, every teacher and every school has a unique story.

The purpose of SchoolOS is not to write these stories, but to help make them visible.
