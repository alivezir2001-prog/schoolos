# Purpose

The Class Overview provides teachers with a meaningful educational understanding of an entire class before teaching begins.

Rather than displaying a simple class list, it highlights educational context, learner needs, classroom patterns and observation opportunities.

Its purpose is to help teachers prepare for learning, not administration.

---

# Vision

Every class has its own character.

Every class has different strengths, challenges and dynamics.

The Class Overview helps teachers understand those dynamics quickly and supports informed educational decisions throughout the school day.

---

# Educational Role

The Class Overview acts as the bridge between the Teacher Dashboard and individual Learner Profiles.

It allows teachers to move from understanding the class as a whole to understanding each learner.

It supports both classroom-level and learner-level educational decision-making.

---

# Educational Questions

Before entering the classroom, the Class Overview should answer:

## 1. What is happening in this class?

A concise AI-generated class summary.

---

## 2. Which learners need my attention today?

Learners who may benefit from additional support, encouragement or observation.

---

## 3. Has anything changed recently?

Recent observations.

New educational evidence.

Important learner developments.

---

## 4. What should I observe today?

Suggested observation opportunities based on recent educational activity.

---

## 5. What strengths does this class currently demonstrate?

Positive classroom patterns such as:

- Collaboration
- Participation
- Responsibility
- Creativity
- Resilience

Educational understanding should always include strengths.

---

# Core Components

## AI Class Brief

A concise educational summary describing the current state of the class.

The summary is generated from recent educational evidence.

---

## Learner Grid

Visual overview of every learner.

Each learner card may display:

- Name
- Photo
- AI status indicator
- Recent observation
- Educational attention indicator

Selecting a learner opens the Learner Card.

---

## Attention Center

Highlights learners requiring educational attention today.

Examples include:

- New learner
- Recent positive development
- Observation opportunity
- Reduced participation
- Important educational alerts

The Attention Center exists to support learning.

Never to label learners.

---

## Recent Educational Activity

Displays recent educational observations affecting the class.

Teachers quickly understand what has changed.

---

## Observation Opportunities

Suggests learners who may benefit from new observations.

The goal is to improve educational understanding rather than increase documentation.

---

## Class Insights

AI-generated educational patterns such as:

- Collaboration trends
- Participation patterns
- Engagement changes
- Emerging classroom strengths
- Areas requiring additional support

Every insight must be supported by educational evidence.

---

# Artificial Intelligence

Artificial Intelligence supports the Class Overview by:

- summarizing classroom dynamics,
- identifying meaningful learner patterns,
- highlighting observation opportunities,
- detecting significant educational changes,
- generating class-level educational insights.

AI supports educational understanding.

Teachers make educational decisions.

---

# Design Principles

The Class Overview should:

- Present understanding before lists.
- Highlight priorities before details.
- Celebrate strengths as well as identify support needs.
- Encourage meaningful observations.
- Explain AI-generated insights.
- Reduce cognitive load.
- Preserve educational continuity.
